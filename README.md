# Retail Sales & Customer Insights Analysis

## Project Overview
This project analyzes retail sales data using SQL in SQLite to uncover trends in revenue, profitability, product performance, customer behavior, and regional sales performance.

## Tools Used
- SQL
- SQLite
- VS Code
- Python

## Business Questions Answered
- What are the overall sales, profit, and profit margin KPIs?
- How do sales and profit trend over time?
- Which products and categories generate the most revenue?
- Which regions perform best?
- Who are the highest-value customers?
- Which sales channels perform best?
- What is the repeat customer rate?
- Which customer segments perform best by quarter?

## Files in This Repository
- `retail_sales_analysis.sql` → SQL queries used for analysis
- `data_generation.py` → Python script used to generate the SQLite database and sample retail data
- `retail_portfolio.db` → SQLite database file
- `screenshots/` → screenshots of query outputs

## Key SQL Skills Demonstrated
- JOINs
- GROUP BY and aggregations
- Common Table Expressions (CTEs)
- Window functions
- KPI analysis
- Customer and product performance analysis

## Example Insights
- Identified top-performing products by revenue and profit
- Compared category-level profit margins
- Evaluated regional sales performance and active customers
- Measured repeat customer rate
- Analyzed quarterly sales trends by customer segment

## How to Run
1. Open the project in VS Code
2. Open the SQLite database file: `retail_portfolio.db`
3. Run the queries in `retail_sales_analysis.sql`
